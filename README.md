Introduction
For Summative Assessment 2, I developed two Android applications using Android Studio and Kotlin. The first app is "Vibe Calculator" – a creative single-activity calculator. The second app is "Study Buddy" – a multiple-view app with data storage using SharedPreferences. Both projects demonstrate the skills learned in the module including layouts, event handling, UI design, and data persistence.
App 1: Vibe Calculator (Single Activity + Modal View)
This app lets users select two different moods and an operation (Blend, Clash, Amplify, Balance). It calculates a creative "Vibe" result with a name, description, emoji, and colour. The result is shown in a custom modal dialog (informative pop-up view). The app uses Spinners, ArrayAdapter, and AlertDialog to meet the brief requirements. No real numbers are calculated — it uses mood combinations creatively.
App 2: Study Buddy (Multiple Views + Data Storage)
This app has three screens: Home, Settings, and Profile. Users can save their name, study goal, and notification preference using SharedPreferences. The data is saved and loaded across app restarts. Navigation between activities uses Intent. This demonstrates multiple views and basic data storage as required.
Technical Skills Demonstrated

Single Activity with modal view (App 1)
Multiple Activities with navigation (App 2)
SharedPreferences for data persistence
Event handling (button clicks, spinner selection)
Modern Kotlin and Constraint Layout
Clean UI design with good colour scheme

Challenges Faced

Creating a creative calculator without using numbers – solved by using mood combinations.
Implementing a proper modal view instead of simple Toast.
Ensuring data persists correctly between activities.

Conclusion
Both apps are fully functional and meet the assessment criteria. This project helped me improve my Android development skills significantly. I am now comfortable with Kotlin, layouts, and data storage in Android Studio.
